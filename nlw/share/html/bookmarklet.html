Search Set Bookmarklet Generator

The upcoming interwiki search will enable searching over one or more workspaces. Initially, we will offer a binary search: either the current workspace, or all the user's workspaces. The user can manually search over a set of workspaces by modifying the query to include multiple workspace names, but this is less than optimal. A GUI tool to build search bookmarklets will make this very easy. Select multiple workspaces from the select list, and watch as the link below updates (both its target and its text). You can drag the link to your bookmark bar and have instant access to a subset of workspaces to search over. [Of course, it doesn't work yet, but that's coming]
